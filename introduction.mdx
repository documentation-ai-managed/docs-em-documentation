## Overview

EM Documentation provides your default space to organize and manage all project documentation. You create structured pages, collaborate with teams, and embed content seamlessly across your projects. Use it to centralize API references, user guides, and changelogs with powerful MDX components for rich, interactive experiences.

## Key Benefits

You gain efficiency with intuitive tools that scale from small projects to enterprise needs. EM Documentation supports version history, custom themes, and integrations for workflows.

Key advantages include:

- **Fast Setup**: Launch your docs space in minutes.
- **Interactive Content**: Engage users with tabs, steps, and code examples.
- **Searchable Knowledge Base**: Full-text search across all pages.

## Getting Started

Follow these steps to set up your first documentation space.

<Steps>
  <Step title="Create a Space" icon="plus">
    Navigate to your dashboard and click "New Space". Enter a name like "Project Docs" and select EM Documentation as the template.
  </Step>
  <Step title="Add Your First Page" icon="file-plus">
    Inside the space, create "introduction.mdx". Use the MDX editor to add frontmatter and content.

````mdx
---
title: My Project
description: Overview of features.
---
````

  </Step>
  <Step title="Publish and Share" icon="share-2">
    Preview changes, then publish. Generate a shareable link or embed code for your site.
  </Step>
</Steps>

## Platform Setup Examples

Choose your preferred method to integrate EM Documentation.

<Tabs>
  <Tab title="Web UI" icon="globe">
    Use the dashboard to manage pages visually. Drag-and-drop components like `<Columns>` for layouts.
  </Tab>
  <Tab title="CLI" icon="terminal">
    Install the CLI and push local MDX files.

<CodeGroup tabs="npm,yarn">
```bash
npm install -g em-docs-cli
em-docs init my-space
```
```bash
yarn global add em-docs-cli
em-docs init my-space
```
</CodeGroup>

  </Tab>
  <Tab title="API" icon="api">
    Create spaces programmatically.

```javascript
const response = await fetch('https://api.em-docs.com/spaces', {
  method: 'POST',
  headers: { 'Authorization': 'Bearer YOUR_API_KEY' },
  body: JSON.stringify({ name: 'My Project Docs' })
});
```

  </Tab>
</Tabs>
