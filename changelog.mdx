## Upgrade Guide

Follow these steps to update your EM Documentation instance.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run this command in your project root to verify your version:

    <CodeGroup tabs="npm,yarn">
    ```bash
    npm list em-docs
    ```
    ```bash
    yarn list em-docs
    ```
    </CodeGroup>
  </Step>

  <Step title="Install Latest" icon="download">
    Update to the newest release:

    <Tabs>
      <Tab title="npm" icon="package">
      ```bash
      npm install em-docs@latest
      ```
      </Tab>
      <Tab title="yarn" icon="package">
      ```bash
      yarn add em-docs@latest
      ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Deploy Changes" icon="rocket">
    Rebuild and redeploy your docs site. Test key pages for compatibility.
  </Step>
</Steps>
